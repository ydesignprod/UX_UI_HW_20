Usability Testing Data
Hypothesis

Veterans with accessibility challenges need a clear application process. Creating an account where they could return to their application and save changes could simplify and streamline the process. Knowing what documents are needed to apply on the first page will allow them to plan for the process and may reduce the number of abandoned, unfinished applications.

Problem Statement

We believe that by creating an easy application process with clearly defined steps, required documents identified early on, and a login portal veterans with accessibility challenges will achieve greater application success and reduce the number of abandoned applications.
